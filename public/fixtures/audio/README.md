# Studio spike audio fixtures

`stem-a.wav` and `stem-b.wav` are deterministic 44.1 kHz, mono, 16-bit PCM WAV files generated by `node scripts/generate-studio-audio-fixtures.mjs`. They contain simple synthesized tones, have no external source material, and may be redistributed with this repository for testing.

Regenerate both files from the repository root after intentionally changing the generator. The two-second fixtures are inputs to the development-only studio spike; fixture URLs are never persisted in the Jam Session manifest.
